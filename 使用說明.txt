============================================
  SwitchBot 排程控制 - 使用說明
============================================

【準備】

1. 電腦需要有藍牙 (一般筆電內建即可)
2. 手機先用 SwitchBot 官方 App 完成下列設定:
   - 加入你的 SwitchBot Bot
   - 進入 Bot 設定 → Mode → 選「Switch Mode」(開關模式)
   - 手機 App 完全關閉 (從多工介面滑掉, 不是按 Home)
   - 或直接把手機開「飛航模式」最保險


【第一次使用】

1. 雙擊 SwitchBotScheduler.exe 啟動程式

2. 程式會自動跳出「掃描 SwitchBot 設備」視窗
   - 按一下 Bot 頂部按鈕喚醒它 (手臂會動一下)
   - 在視窗中點「重新掃描」
   - 從清單選你的 Bot (名稱為 WoHand)
   - 點「選擇」

3. 在「手動測試」區塊點「開啟」或「關閉」
   確認 Bot 有動作後, 就可以開始設定排程


【設定排程】

在「排程」區塊點「新增」, 設定:
  - 時間 (24 小時制, 例 08:00 或 23:30)
  - 動作 (開啟 / 關閉 / 按壓)
  - 星期 (可複選, 例如只有週一到週五)
  - 啟用 (打勾表示會執行; 取消打勾等於暫時停用但不刪除)

點「確定」即加入排程. 支援:
  - 新增: 點「新增」
  - 修改: 選一筆 → 點「編輯」(或直接雙擊該筆)
  - 刪除: 選一筆 → 點「刪除」
  - 暫停: 選一筆 → 點「啟用/停用」(V 表示啟用中)


【執行方式 (背景模式)】

排程在程式執行期間會自動執行.

按右上角 X 不會結束程式!
  → 視窗會「縮到右下角系統匣」(時鐘旁的紅色小圖示),
    程式繼續在背景執行, 排程不會中斷
  → 雙擊系統匣的紅色圖示, 可重新開啟主視窗
  → 要「真正結束」程式: 在系統匣圖示上按右鍵 → 選「結束程式」

提示: 若在系統匣看不到圖示, 點時鐘旁的「^」箭頭展開隱藏圖示.
建議勾選左下角「開機自動啟動」, 每次 Windows 開機都會自動啟動.


【開機自動啟動】

勾選左下角「開機自動啟動」後:
  - 下次 Windows 開機會自動啟動此程式
  - 要取消就把勾取消掉

若要暫時不讓它開機啟動, 取消勾選即可.
設定檔儲存在: %APPDATA%\SwitchBotScheduler\config.json


【故障排除】

◆ 掃描不到 Bot
  1) 手機 SwitchBot App 是不是完全關閉? (最常見原因)
     - 建議直接開手機飛航模式測試
  2) 按一下 Bot 頂部按鈕喚醒它, 然後立刻重新掃描
  3) Bot 距離電腦 5 公尺內
  4) 電腦藍牙是否開啟: 設定 → 藍牙與裝置

◆ 手動測試 / 排程顯示失敗
  - 通常是藍牙連線瞬斷, 等 5-10 秒再試
  - 如果連續失敗: 檢查手機 App 是否又搶回連線
  - 檢查 Bot 電池電量 (CR2 鋰電池)

◆ Bot 有回應但電器沒切換
  - 確認手機 App 把 Bot 設為「Switch Mode」
  - Press Mode 只會按一下, 不會有獨立的開關狀態

◆ 程式打不開 / Windows 警告
  - Windows SmartScreen 可能擋下不認識的 exe
  - 點「其他資訊」→「仍要執行」
  - 或在 Windows 安全性 → 病毒與威脅防護中加入例外

◆ 電腦休眠後排程不動
  - Windows 睡眠/休眠狀態下程式不會執行
  - 建議到「電源選項」把睡眠設為「永不」


【移除程式】

1. 左下角取消勾選「開機自動啟動」
2. 在右下角系統匣圖示上按右鍵 → 選「結束程式」
3. 刪除 SwitchBotScheduler.exe
4. 設定檔如需一併清除, 刪除: %APPDATA%\SwitchBotScheduler\
